# Prodigy_03_Text-Generation-with-Markov-Chains

## Objective
Generate text using Markov Chains by predicting the next word based on previously observed word sequences.

## Technologies Used
- Python
- Markov Chains
- Random Library

## Installation

pip install -r requirements.txt

## Run

python text_generation_markov.py

## Features
- Text generation using Markov Chains
- Simple probabilistic language model
- Randomized text creation
- Easy implementation

## Output
Generates new text based on training data provided in the program.
